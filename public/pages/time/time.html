Time in JavaScript

There are several ways to work with dates in JavaScript and they all have different purposes and provides different functionalities. Some of the main ways are using new Date(), this a build in JavaScript Date object that represents the current data and time. It can also accept arguments to create a Date object for a specific data an time `new Data(1990-04-18)`. But if no arguments are given it will just return the current date.

The date and time is represented as an ISO 8601 formatted date and time string in UTC (Coordinated Universal Time). How to read it:

  • 2024-02-23: This part represents the date in the format YYYY-MM-DD. In this case, it's February 23, 2024.
  • T: The uppercase "T" is a separator that indicates the beginning of the time component.
  • 08:22:51.979: This part represents the time in the format HH:mm:ss.SSS. In this case, it's 08 hours, 22 minutes, 51 seconds, and 979 milliseconds.
  • Z: The uppercase "Z" denotes that the time is in UTC (Coordinated Universal Time), also known as GMT (Greenwich Mean Time). The "Z" stands for "Zulu time," which is a term used in aviation and military contexts to refer to UTC time.

Another way is using Date.now(), the method returns a numeric value corresponding to the time in millieseconds since the Unix Epoch (January 1, 1970, UTC) .
